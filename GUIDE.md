# Aedis - Guide Simple

## Qu'est-ce que Aedis ?

**Aedis** est une application web qui transforme vos plans architecturaux en pièces meublées photoréalistes grâce à l'intelligence artificielle. 

En quelques clics, vous pouvez :
- Télécharger un plan de votre pièce
- Décrire le style de décoration souhaité
- Obtenir une vue 360° photoréaliste de votre pièce meublée
- Cliquer sur les meubles pour trouver des produits à acheter en ligne

## Comment ça fonctionne ?

### 1. **Téléchargement du plan**
Vous commencez par télécharger une photo ou un plan de votre pièce (format JPG, PNG ou WEBP). Si vous n'avez pas de plan, vous pouvez utiliser les images de test fournies.

### 2. **Description du style**
Vous décrivez le style de décoration que vous souhaitez : "Minimaliste moderne", "Scandinave cosy", "Loft industriel", etc. Vous pouvez aussi être plus précis dans votre description.

### 3. **Génération par l'IA**
L'intelligence artificielle analyse votre plan et génère une vue 360° photoréaliste de la pièce meublée selon votre style. Cette étape prend quelques secondes.

### 4. **Exploration et shopping**
Une fois la pièce générée, vous pouvez :
- **Explorer** : Faire glisser votre souris pour tourner autour de la pièce en 360°
- **Sélectionner** : Cliquer sur "Sélectionner" puis dessiner un rectangle autour d'un meuble qui vous plaît
- **Acheter** : Consulter une liste de produits similaires disponibles en ligne avec leurs prix et liens d'achat





## Technologies utilisées

Aedis utilise des technologies modernes pour offrir une expérience fluide :

- **Intelligence Artificielle** : Pour générer les images photoréalistes (Google Gemini et OpenAI)
- **Recherche visuelle** : Pour trouver des produits similaires aux meubles sélectionnés
- **Vue 360°** : Pour explorer la pièce comme si vous y étiez
- **Design moderne** : Interface élégante avec des effets de verre liquide

## Pour qui ?

Aedis s'adresse à :
- **Particuliers** qui souhaitent visualiser leur futur intérieur avant de meubler
- **Architectes** qui veulent présenter leurs projets de manière immersive
- **Décorateurs** qui cherchent des idées et des produits pour leurs clients
- **Toute personne** curieuse de voir à quoi pourrait ressembler une pièce avec différents styles

## Comment utiliser Aedis ?

1. Ouvrez le site web Aedis
2. Cliquez sur la zone de téléchargement ou utilisez une image de test
3. Entrez votre style de décoration (ex: "Scandinave cosy")
4. Cliquez sur "Générer"
5. Attendez quelques secondes
6. Explorez votre pièce en 360°
7. Sélectionnez un meuble pour voir les produits disponibles

C'est aussi simple que ça !
